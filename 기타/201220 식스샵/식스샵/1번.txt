문제 설명
g-index는 연구자의 연구 업적을 평가하는 지표의 하나입니다. 어떤 연구자가 발표한 논문의 개수가 N일 때 g-index는 다음과 같습니다.

g-index : N개의 논문 중 상위 g개의 논문의 인용 횟수의 합이 g2 이상이 되도록 하는 g 값 중에서 최댓값
어떤 연구자가 발표한 N개의 논문의 각각의 인용 횟수 K가 담긴 배열이 매개변수 paper로 주어질 때, 이 연구자의 g-index를 return 하도록 solution 함수를 완성해 주세요.

제한사항
논문의 개수 N : N은 1,000이하의 자연수
각 논문의 인용 횟수 K: K는 0 이상 10,000 이하의 정수
입출력 예
paper	result
[1, 0, 0, 3, 0, 1]	2
[7, 5, 8, 10, 6, 9, 5]	7
[3, 0, 3, 0, 3, 0]	3
입출력 예 설명
입출력 예 #1
상위 2개의 논문의 인용 횟수의 합이 4로 22 이상이고, 상위 3개의 논문의 인용횟수의 합은 5로 32 보다 작으므로 g-index는 2입니다.

입출력 예 #2
상위 7개의 논문의 인용 횟수의 합이 50으로 72 이상이고, 논문의 개수는 최대 7개 이므로 g-index는 7입니다.

입출력 예 #3
상위 3개의 논문의 인용 횟수의 합이 9로 32 이상이고, 상위 4개의 논문의 인용횟수의 합은 9로 42 보다 작으므로 g-index는 3입니다.
